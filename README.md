
# Odkryj Polskę (Discover Poland)


## Description

Odkryj Polskę is a website designed to showcase the beauty, history, and attractions of Poland. It provides users with an interactive experience to explore scenic landscapes, historical landmarks, and unique locations across the country.


## Overview

This website is built with HTML and CSS and features a clean, responsive layout. The design focuses on accessibility and visual appeal, with sections highlighting Poland's history, nature, and unique destinations.

## Features
- Responsive Header: Includes navigation links to all main pages.

- Main Sections:

  - Introduction: Highlights the beauty of Poland with an invitation to explore.

  - Why Visit Poland: Displays key reasons to visit, including history, nature, and unique places, each with a descriptive icon.

  - Call to Action: Encourages users to browse the gallery or contact the team.

- Footer: Contains a short description of the site, quick links, and team members.

- Gallery Links: Direct links to explore images of Poland.

## Pages
- index.php – Homepage

- galeria.php – Gallery of images and destinations

- opolsce.php – Information about Poland

- atrakcje.php – Attractions across the country

- kontakt.php – Contact form and information

## Usage
1. Clone or download the repository.

2. Open index.php in a web browser to view the homepage.

3. Navigate through the site using the header links.

## Database Note
If any pages use a database (for example galeria.php, atrakcje.php, or kontakt.php), make sure to update the database name at the top of each page before running the site to ensure proper connection.
